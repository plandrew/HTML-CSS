Types of containers:

  • inline
  • block

--- Padding 50px from the left / This is content inside inline element
  • Padding used to move the content inside the container
  • No margin
  • Images as a standard are inline elements
  • Padding from the left to move the content 50px from the left side
  • Example tags = span, p, img
--- Margin 50px from the left / --- Padding 50px from the left / This is content inside box element
  • Take all available width
  • Padding used to move the content inside the container
  • Margin used to posit the element inside the viewport
  • Example tags = div, section, main, navbar, header, footer
